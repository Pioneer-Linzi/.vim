# vim config file .vimrc
	.vimrc  vim 第一配置文件 里面配置了分配置文件的路径
	.vimrc.config 	vim的配置文件，
	.vimrc.bundle 插件的配置文件，主要是插件
	.vimrc.bundle.config vim 插件的配置文件，主要有快捷键，与插件的配置
